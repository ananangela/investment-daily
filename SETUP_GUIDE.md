# 投資日報 - 自動化系統設置指南

## 📋 系統概述

**投資日報** 是一個每日自動更新的台股投資新聞網站，由 Claude AI 自動搜尋、分析、撰寫。

- 🌐 **網站地址**: https://ananangela.github.io/investment-daily/
- 📅 **更新頻率**: 每天早上 8 點（台北時間）
- 💰 **成本**: 完全免費
- 🤖 **自動化**: 完全自動（無需手動操作）

---

```

↓

GitHub Pages 自動部署
└── 網站自動更新，無需重新構建
```

## 📁 檔案說明

### `index.html`
- 主網頁文件，包含所有 UI 和 JavaScript
- 特點:
  - 動態讀取 `articles.json`
  - 響應式設計
  - 支援日期導航、分類篩選、可展開名詞卡片

### `articles.json`
- 所有文章數據的中央存儲
- 格式: 以日期為 key 的 JSON 物件
- 每篇文章包含:
  - `headline`: 標題
  - `deck`: 副標題
  - `source`: 新聞來源
  - `sourceUrl`: 原文連結
  - `summary`: 詳細摘要
  - `keypoints`: 重點列表
  - `terms`: 名詞解釋
  - `tags`: 分類標籤
  - `difficulty`: 難度等級

## 🔄 自動化流程

### 每天 08:00 (台北時間)

#### 1️⃣ 搜尋新聞
- 使用 WebSearch 尋找最新台股投資新聞
- 重點關鍵字:
  - AI 晶片、半導體
  - 台積電、聯發科等權值股
  - ETF、高息產品
  - 其他重要個股/產業

#### 2️⃣ 選擇文章
挑選最適合初學者的 1 篇文章，標準:
- ✓ 新聞價值高、有實際數據
- ✓ 有明確來源網站
- ✓ 內容有教育意義
- ✓ 適合初學者理解

#### 3️⃣ 分析文章
- 撰寫清晰的繁體中文摘要（150-200 字）
- 提取 3-5 個最重要的重點
- 找出 3-5 個關鍵投資名詞並詳細解釋
- 判斷難度等級（初級/進階）

#### 4️⃣ 生成 JSON
- 以今天的日期 (YYYY-MM-DD) 為 key
- 產生符合 `articles.json` 格式的數據

#### 5️⃣ 更新網站
- 將新數據添加到 `articles.json`
- Git commit 並 push 到 GitHub
- GitHub Pages 自動部署（幾秒內生效）

## ⚙️ 維護和故障排查

### 常見問題

**Q: 為什麼網站沒有更新？**
- A: 檢查 articles.json 是否已推送到 GitHub
- 可能需要 5-10 秒才能在 GitHub Pages 上生效
- 嘗試強制刷新 (Ctrl+Shift+R 或 Cmd+Shift+R)

**Q: 我可以手動編輯 articles.json 嗎？**
- A: 可以！直接編輯 articles.json 並推送，網站會自動更新
- 但建議保持 JSON 格式的一致性

**Q: 如何修改自動化時間？**
- A: 需要在 Claude chat 中重新設置 CronCreate 任務
- 當前設置: 每天早上 8 點（台北時間）

**Q: 文章來源連結失效了怎麼辦？**
- A: 手動編輯 articles.json 中的 `sourceUrl`
- 或等待下一次自動更新時修正
